from flask import Flask, render_template, request, jsonify
import json
import os
from datetime import datetime

app = Flask(__name__)

# Simple in-memory storage (in production, use a database)
data_file = 'step_tracker_data.json'

def load_data():
    if os.path.exists(data_file):
        with open(data_file, 'r') as f:
            return json.load(f)
    return {}

def save_data(data):
    with open(data_file, 'w') as f:
        json.dump(data, f, indent=2)

@app.route('/')
def index():
    data = load_data()
    return render_template('index.html', data=data)

@app.route('/add_entry', methods=['POST'])
def add_entry():
    data = load_data()
    date = request.form.get('date', datetime.now().strftime('%Y-%m-%d'))
    
    if date not in data:
        data[date] = {
            'steps': 0,
            'calories': 0,
            'water_ml': 0
        }
    
    data[date]['steps'] = int(request.form.get('steps', 0))
    data[date]['calories'] = int(request.form.get('calories', 0))
    data[date]['water_ml'] = int(request.form.get('water_ml', 0))
    
    save_data(data)
    return jsonify({'success': True, 'message': 'Entry saved!'}), 201

@app.route('/get_data', methods=['GET'])
def get_data():
    data = load_data()
    return jsonify(data)

@app.route('/delete_entry/<date>', methods=['DELETE'])
def delete_entry(date):
    data = load_data()
    if date in data:
        del data[date]
        save_data(data)
        return jsonify({'success': True, 'message': 'Entry deleted!'}), 200
    return jsonify({'success': False, 'message': 'Date not found'}), 404

if __name__ == '__main__':
    app.run(debug=True)
```

# Step Tracker Flask App

A minimal Flask web application for tracking daily steps, calories burned, and water intake.

## Features

- Track daily steps
- Log calories burned
- Monitor water intake (in ml)
- Simple, clean UI
- Data persists in JSON file

## Files Created

| File | Description |
|------|-------------|
| `app.py` | Main Flask application |
| `templates/index.html` | Web interface template |
| `requirements.txt` | Python dependencies |
| `README.md` | This documentation |

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   python app.py
   ```

3. Open your browser to `http://localhost:5000`

## Data Storage

Data is stored in `step_tracker_data.json` in the workspace directory.

## API Endpoints

- `GET /` - View the tracker interface
- `POST /add_entry` - Add/update a daily entry
- `GET /get_data` - Get all data
- `DELETE /delete_entry/<date>` - Delete an entry

## Usage

1. Enter a date (defaults to today)
2. Fill in steps, calories, and water intake
3. Click "Save Entry"
4. View your history below the form

## Notes

- This is a minimal app for demonstration purposes
- Data is stored in a JSON file (not a database)
- For production use, consider adding a proper database
