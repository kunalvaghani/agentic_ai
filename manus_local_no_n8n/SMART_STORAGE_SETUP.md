Smart storage upgrade

What changed
- Added agent/storage_manager.py
- Screenshots now default to storage/screenshots/... when no path is provided
- Browser text captures can be saved automatically
- Added storage tools: storage_list_recent, storage_save_text, storage_organize_file
- Bridge now auto-saves final run summaries into storage/documents/...
- Bridge recent artifacts now include storage files

How to use
1. Replace your project files with this package or copy the changed files into your existing project.
2. Restart the bridge starter BAT.
3. In Open WebUI, keep using the Manus Local Bridge model.

Examples
- "Open YouTube, search for lofi Bollywood songs, take a screenshot, and save the visible page text."
- "Take a desktop screenshot and save a short note explaining what happened."
- "Organize README.md into storage with a better name."
